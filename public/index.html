Universal bots

DeepSeek AI

Global DeepSeek settings

Enable DeepSeek AI bots
DeepSeek в этой версии работает только как text-only провайдер. OCR, vision, audio/STT и native hybrid memory не включены, потому что в официальном API для них нет подтверждённой документации.

DeepSeek bots and persona

Bot avatar The image is stored separately from JSON persona settings.
Выбирайте `deepseek-chat` для обычного текстового режима и `deepseek-reasoner` для reasoning-сценариев. Hybrid memory для DeepSeek отключён.

DeepSeek bots in chats

DeepSeek-боты отвечают только на @mention или reply. Hybrid memory для них недоступен, поэтому здесь всегда используется только simple context.

AI-Яндекс

Global Yandex settings

Enable Yandex AI bots
Идентификатор каталога вводится в поле Folder ID выше. Он нужен для сборки modelUri вида gpt://<folder_ID>/yandexgpt/latest.

Yandex bots and persona

Bot avatar The image is stored separately from JSON persona settings.
Yandex hybrid memory uses separate embedding tables and does not touch OpenAI memory.

Yandex bots in chats

Yandex bots answer only to @mention or reply to that bot.

Grok text bots

Bot avatar The image is stored separately from JSON persona settings.
Grok hybrid memory uses separate Grok vector tables and does not touch OpenAI or Yandex memory.

Grok image bots

Image bot avatar The image is stored separately from JSON persona settings.
Image bots always use simple mode and answer with one generated image saved to local uploads.

Grok universal bots

Universal bot avatar The image is stored separately from JSON persona settings.
Universal Grok bots can answer with text, analyze images, generate new images, and edit a supported source image.
